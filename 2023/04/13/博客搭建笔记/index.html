基于Hexo，Git 利用Gituhb 搭建的Blog 的流程。

  1. 安装Node.js.以及Git
  2. Go to github, <用户名>.github.io，然后点击Create repository
  3. win+R 运行 cmd.exe More info: 创建一个blog

部署blog

1.安装hexo-deployer-git

修改根目录下的 _config.yml，配置 GitHub相关信息

deploy:
type: git
repo:
github:
url: https://github.com/aaaa/bbb.git e.g.https://github.com/FelixJian9/FelixJian9.github.io
branch: main
branch: main
token: ‘’ #

以及

# URL
## Set your site url here. For example, if you use GitHub Page, set url as 'https://username.github.io/project'
url: https://FelixJian9.github.io
permalink: :year/:month/:day/:title/
permalink_defaults:
pretty_urls:
trailing_index: true # Set to false to remove trailing 'index.html' from permalinks
trailing_html: true # Set to false to remove trailing '.html' from permalinks

插入图片

使用png格式，jpg格式有图片大小限制
More info: 插入图片
居中命令：
<center>

改变大小：

1.找到对应css文件 e.g. E:\blog\themes\frame\source\css\post\media.styl 以记事本打开
2. 创建图片尺寸缩放分区 e.g. 50% 缩放
在

img
width 100%
height 100%

后加入分区

.image50
img
width 50%
height 50%

3.在blog正文写入图片的分区命令

<div class="image50">
加入插入图片命令e.g. {% asset_img 2.jpg COVID-19test%}
完成外包</div>

4.完成

对feature图片大小进行调整

1.找到对应css文件 e.g. E:\blog\themes\frame\source\css\post\post_head.styl 以记事本打开
2.添加缩放命令在 .post-img 后

   width 50%
    height 50%

3.完成
